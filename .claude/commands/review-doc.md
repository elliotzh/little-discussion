## 审阅维度

### 1. 论证逻辑
- 核心论点是否清晰？
- 论证链条是否完整、自洽？
- 是否存在逻辑跳跃或隐含假设未被说明？

### 2. 论据充分性
- 关键论断是否有足够支撑？
- 是否存在需要补充数据、案例或引用的地方？
- 反面观点是否被充分考虑？

### 3. 结构与层次
- 章节划分是否合理？
- 段落间的过渡是否自然？
- 整体叙事节奏是否恰当？

### 4. 语言质量
- 表达是否准确、流畅？
- 是否存在冗余或模糊的表述？
- 术语使用是否一致？
